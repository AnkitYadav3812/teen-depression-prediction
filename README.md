# Teen Depression Prediction System

A Machine Learning project that predicts depression risk in teenagers using social media usage and lifestyle factors.

## Features

- Random Forest Classifier
- SMOTE for class balancing
- Streamlit Web Application
- Depression Probability Score
- Low / Medium / High Risk Classification
